## 查看文件占用

```shell
# 查看文件占用
lsof | grep 1.txt
# 查看进程占用 通过端口号或进程名
sudo lsof -p 23039
sudo lsof -c redis-server
```

## 查看进程

```shell
ps -ef | grep redis
# 查看CPU和内存占用
ps -aux | grep redis
kill -9 [id]
pkill -9 redis-server
```

## 查看端口

```shell
netstat -tunple | grep 6379
netstat -tunple | grep redis
netstat -na | grep 6379
```
