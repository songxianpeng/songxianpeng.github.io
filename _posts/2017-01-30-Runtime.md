## Runtime

每个 Java 应用程序都有一个 Runtime 类实例，使应用程序能够与其运行的环境相连接。

```java
public static void main(String[] args) throws IOException {
	Runtime r = Runtime.getRuntime();
	r.exec("winmine");// 扫雷
	r.exec("notepad");
	r.exec("calc");
	r.exec("shutdown -s -t 10000");// cmd命令
	r.exec("shutdown -a");
}

```
